# Audience Persona, Output Template

This is the full output template referenced by Step 5 of the Audience Persona Builder skill.

The order matters: JTBD and behaviour open the persona, demographics support, 5 Rings of Buying Insight structure the buying decision, modern-practice fields close it out.

Each field carries its own confidence marker (H / M / L) and source (Maddie's observation, research, or both). A field-level marker is more useful than one rating across the whole persona, because some claims are strong and some are inferred.
