# Visual Brand Check, Worked Examples

Three worked examples for [[Visual Brand Check]]. Demonstrates how the skill runs against a stub-guidelines asset (Group A + C only), a locked-guidelines campaign set, and a heavy-revision case.

---

## Example 1, Stub guidelines, Group A + C only

```
Maddie: visual check this for Fade Golf [drops Canva export of a membership reel cover]

Skill: Brand Fade Golf confirmed. Asset type: vertical 1080x1920, looks like an IG reel cover. Standalone or part of a set?

Maddie: standalone

Skill: Brand Guidelines for Fade Golf are still a stub. I can run Group A (visual layout) and Group C (accessibility and platform fit). Group B (brand adherence) needs the guidelines locked. Run Group A + C, or pause to /learning the guidelines first?

Maddie: A + C, do not pause

Skill: [Runs Group A + Group C]

Verdict: Minor fixes

## Group A: Visual Layout

| # | Finding | Confidence | Suggested fix |
|---|---|---|---|
| A1 | Caption text at the bottom is small for a vertical scroll-stop format, and uses a Light weight that thins out on mobile | High | Increase caption size to at least 1.5x and bump weight from Light to Regular or Medium |
| A4 | Z-pattern implied (headline top-left, hero centre, CTA bottom-right) but headline and hero compete for the first beat | Medium | Reduce headline weight or contrast slightly so the hero leads and headline supports |
| A7 | Bottom 8% of the asset risks overlap with IG reel UI (like + share buttons); CTA currently sits in that zone | High | Move the CTA up 8 to 10% to clear the UI safe zone and land in the thumb-reachable area |
| A8 | CTA position currently in bottom-right corner past the reel UI, putting it just outside the easy thumb-reach for a portrait phone | Medium | Move CTA up and slightly centred |

## Group C: Accessibility and Platform Fit

| # | Finding | Confidence | Suggested fix |
|---|---|---|---|
| C1 | White caption over a busy course photo with bright sky, contrast likely below 4.5:1 in the brightest patches | High | Add a semi-transparent dark scrim behind caption, or move caption onto a solid background block |
| C2 | Alt text reminder | Reminder | Confirm alt text is set on the reel before publishing |

## Top 3 fixes
1. Resolve C1 contrast risk on caption over photo (accessibility fail risk)
2. Move CTA above the reel UI safe zone and into thumb-reachable area
3. Increase caption size and weight for mobile legibility

## Summary
Three priority fixes. C1 is the accessibility risk worth addressing first. Group B not assessed (guidelines stub).
```

## Example 2, Guidelines locked, full check on campaign set

```
Maddie: visual check the orthotics carousel for Stride [drops slide 1 of 5]

Skill: Brand Stride confirmed. Asset type: 1080x1080 IG feed, slide 1 of a carousel. Brand Guidelines: locked. Want me to review just this slide, or drop the other 4 so I can check cross-asset consistency too?

Maddie: here are all 5 [drops slides 2 to 5]

Skill: Reviewing the full set. [Runs all groups + campaign set check]

Verdict: Minor fixes

## Group A: Visual Layout

| # | Finding | Confidence | Suggested fix |
|---|---|---|---|
| A3 | Slide 3 has tighter top margin than slides 1, 2, 4, 5 (whitespace breaks the rhythm of the set) | High | Match slide 3 top margin to the other four |

## Group B: Brand Adherence

| # | Finding | Confidence | Suggested fix |
|---|---|---|---|
| B1 | Navy + ice-blue palette consistent across all 5 slides | Medium-high | OK |
| B2 | Brand serif used for headlines on every slide | Medium | OK |
| B3 | Logo present and consistent across slides | High | OK |
| B4 | Real patient photos, not stock | Medium | OK, confirm patient consent is on file before posting |
| B5 | All 5 slides show younger athletes (25 to 45 range). Secondary persona is 60+ diabetic, not represented in this set | Medium | Acceptable if this carousel is targeted at primary persona only. If it is the brand-level orthotics intro, add a slide that includes the secondary audience |

## Group C: Accessibility and Platform Fit

| # | Finding | Confidence | Suggested fix |
|---|---|---|---|
| C1 | Navy body text on ice-blue background on slides 2 and 4 sits at roughly 4:1 by visual estimate, borderline below the 4.5:1 normal-text threshold | Medium | Either darken the text to near-black or lighten the background slightly. Verify with WebAIM Contrast Checker before publishing |
| C2 | Alt text reminder | Reminder | Set alt text on the carousel post |

## Campaign / Set Consistency

| # | Finding | Confidence | Suggested fix |
|---|---|---|---|
| CS1 | Palette consistent across all 5 slides | High | OK |
| CS2 | Typography hierarchy consistent (headline weight + body weight same across slides) | High | OK |
| CS3 | Imagery treatment consistent (same warm-toned natural-light grading on slides 1 to 4; slide 5 reads slightly cooler) | Medium | Re-grade slide 5 to match the warmer tone of slides 1 to 4 |
| CS4 | Logo position and size consistent across set | High | OK |
| CS5 | Message hierarchy consistent (each slide leads with a question or pain-point headline, then short body) | High | OK |

## Top 3 fixes
1. C1 contrast borderline on slides 2 and 4, verify with WebAIM and adjust if needed
2. CS3 slide 5 grading inconsistency vs rest of set
3. B5 representation note for Maddie to confirm scope of this carousel

## Summary
Strong set overall. One accessibility risk to verify, one imagery-tone inconsistency on slide 5, one representation note for Maddie to confirm scope.
```

## Example 3, Heavy revision

```
Maddie: design check this F&E flyer [drops a draft]

Skill: Brand F&E confirmed. Asset type: A5 portrait, looks like a leaflet front. Guidelines: locked.

Verdict: Heavy revision

## Group A: Visual Layout

| # | Finding | Confidence | Suggested fix |
|---|---|---|---|
| A1 | Body text very small | High | Increase from estimated ~9pt to 12pt+ for A5 print |
| A2 | Three different alignments in one panel (headline centred, subhead left, body text right) | High | Pick one alignment for the panel and apply consistently |
| A3 | Margins uneven, top is 15mm but sides 5mm | High | Even margins to minimum 12mm all round |
| A4 | No clear visual entry point | High | Increase headline weight; or add a hero image |

## Group B: Brand Adherence

| # | Finding | Confidence | Suggested fix |
|---|---|---|---|
| B1 | Background uses a default Canva pink not in F&E palette | High | Replace with F&E warm-neutral from guidelines |
| B2 | Typography uses Canva default sans-serif, not F&E brand serif | Medium | Switch to brand serif for headlines, keep sans for body if guidelines allow |
| B3 | Logo too small and too close to edge (clear-space violation) | High | Increase logo size to minimum per guidelines, increase clear space to 1.5x logo height |

## Top 3 fixes
1. Replace background colour with F&E palette
2. Increase body text size for print legibility
3. Fix logo size + clear space

## Summary
Heavy revision needed. Recommend rebuilding from the F&E print template (in guidelines) rather than patching this draft.
```
